# MARKIE

MARKIE is a React based Real-Time Markdown editor made with showdownjs.

## Getting Started

```
git clone https://github.com/akhilmhdh/markie.git
npm install 
npm start
```

### Prerequisites

```
Node
npm 
```

## Built With

* [react](https://reactjs.org/) - The web framework used
